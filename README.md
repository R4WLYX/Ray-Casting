# Ray-Casting
Simple ray-casting engine in c++

# Movement
W - Forwards<br>
S - Backwards<br>
A - Strafe Left<br>
D - Strafe Right<br>
Q - Turn Left<br>
E - Turn Right<br>
Shift - Sprint
